# CROSS_SECTION/QUOTES_FEATURE：69 个归一化截面特征

代码位置：
- `data_preprocess/operator_futures/cross_section/create_feature.py`（入口）
- `data_preprocess/operator_futures/cross_section/base_feature_util.py`（核心函数 `process_quotes_n_feature` + `normalize_feature_cross_section`）

输入为 `BASE_FEATURE` 的 112 列。先通过正则 `r"^(.*?)(open|high|low|close|twap|awap|vwap)(.*)$"` 去除 OHLC 组列，再对剩余列按 buy/sell、up/down/flat、bid/ask 三种模式分组归一化。

---

## 统一记号

- `ε = 1e-15`：防零常数（代码中为 `minium`）
- 归一化公式中的 `all` 表示该组的总量列，`buy/sell`、`up/down/flat`、`bid/ask` 表示子分量列

---

## 一、Buy/Sell 归一化（18 列）

对包含 buy 和 sell 子分量的列组，以总量 `all` 为基准做归一化。

每组格式：`[all, buy, sell]`，生成 3 列：
- `{all}_buy_bsnorm = buy / (all + ε)`
- `{all}_sell_bsnorm = sell / (all + ε)`
- `{all}_buysell_imbalance_bsnorm = (buy - sell) / (all + ε)`

| # | 列名 | 含义 | 计算公式 |
|---|------|------|----------|
| 1 | `volume_buy_bsnorm` | buy 方向成交量占比 | `volume_buy / (volume + ε)` |
| 2 | `volume_sell_bsnorm` | sell 方向成交量占比 | `volume_sell / (volume + ε)` |
| 3 | `volume_buysell_imbalance_bsnorm` | 买卖成交量不平衡度 | `(volume_buy - volume_sell) / (volume + ε)` |
| 4 | `tradeval_buy_bsnorm` | buy 方向成交金额占比 | `tradeval_buy / (tradeval + ε)` |
| 5 | `tradeval_sell_bsnorm` | sell 方向成交金额占比 | `tradeval_sell / (tradeval + ε)` |
| 6 | `tradeval_buysell_imbalance_bsnorm` | 买卖成交金额不平衡度 | `(tradeval_buy - tradeval_sell) / (tradeval + ε)` |
| 7 | `ntrade_buy_bsnorm` | buy 方向成交笔数占比 | `ntrade_buy / (ntrade + ε)` |
| 8 | `ntrade_sell_bsnorm` | sell 方向成交笔数占比 | `ntrade_sell / (ntrade + ε)` |
| 9 | `ntrade_buysell_imbalance_bsnorm` | 买卖成交笔数不平衡度 | `(ntrade_buy - ntrade_sell) / (ntrade + ε)` |
| 10 | `ntrade_up_buy_bsnorm` | buy 方向上涨笔数占全方向上涨笔数比 | `ntrade_up_buy / (ntrade_up + ε)` |
| 11 | `ntrade_up_sell_bsnorm` | sell 方向上涨笔数占全方向上涨笔数比 | `ntrade_up_sell / (ntrade_up + ε)` |
| 12 | `ntrade_up_buysell_imbalance_bsnorm` | 买卖上涨笔数不平衡度 | `(ntrade_up_buy - ntrade_up_sell) / (ntrade_up + ε)` |
| 13 | `ntrade_down_buy_bsnorm` | buy 方向下跌笔数占全方向下跌笔数比 | `ntrade_down_buy / (ntrade_down + ε)` |
| 14 | `ntrade_down_sell_bsnorm` | sell 方向下跌笔数占全方向下跌笔数比 | `ntrade_down_sell / (ntrade_down + ε)` |
| 15 | `ntrade_down_buysell_imbalance_bsnorm` | 买卖下跌笔数不平衡度 | `(ntrade_down_buy - ntrade_down_sell) / (ntrade_down + ε)` |
| 16 | `ntrade_flat_buy_bsnorm` | buy 方向不变笔数占全方向不变笔数比 | `ntrade_flat_buy / (ntrade_flat + ε)` |
| 17 | `ntrade_flat_sell_bsnorm` | sell 方向不变笔数占全方向不变笔数比 | `ntrade_flat_sell / (ntrade_flat + ε)` |
| 18 | `ntrade_flat_buysell_imbalance_bsnorm` | 买卖不变笔数不平衡度 | `(ntrade_flat_buy - ntrade_flat_sell) / (ntrade_flat + ε)` |

## 二、Up/Down/Flat 归一化（33 列）

对包含 up、down、flat 子分量的列组，以总量 `all` 为基准做归一化。

### 2.1 四分量组 `[all, up, down, flat]`（3 组 × 5 列 = 15 列）

每组生成 5 列：
- `{all}_up_udnorm = up / (all + ε)`
- `{all}_down_udnorm = down / (all + ε)`
- `{all}_flat_udnorm = flat / (all + ε)`
- `{all}_updown_imbalance_udnorm = (up - down) / (all + ε)`
- `{all}_updownflat_vol_udnorm = (up + down - flat) / (all + ε)`

#### 组 1：全方向成交涨跌统计 [ntrade, ntrade_up, ntrade_down, ntrade_flat]

| # | 列名 | 含义 | 计算公式 |
|---|------|------|----------|
| 19 | `ntrade_up_udnorm` | 上涨笔数占比 | `ntrade_up / (ntrade + ε)` |
| 20 | `ntrade_down_udnorm` | 下跌笔数占比 | `ntrade_down / (ntrade + ε)` |
| 21 | `ntrade_flat_udnorm` | 不变笔数占比 | `ntrade_flat / (ntrade + ε)` |
| 22 | `ntrade_updown_imbalance_udnorm` | 涨跌笔数不平衡度 | `(ntrade_up - ntrade_down) / (ntrade + ε)` |
| 23 | `ntrade_updownflat_vol_udnorm` | 涨跌活跃度（涨+跌-不变占比） | `(ntrade_up + ntrade_down - ntrade_flat) / (ntrade + ε)` |

#### 组 2：buy 方向成交涨跌统计 [ntrade_buy, ntrade_up_buy, ntrade_down_buy, ntrade_flat_buy]

| # | 列名 | 含义 | 计算公式 |
|---|------|------|----------|
| 24 | `ntrade_buy_up_udnorm` | buy 上涨笔数占 buy 总笔数比 | `ntrade_up_buy / (ntrade_buy + ε)` |
| 25 | `ntrade_buy_down_udnorm` | buy 下跌笔数占 buy 总笔数比 | `ntrade_down_buy / (ntrade_buy + ε)` |
| 26 | `ntrade_buy_flat_udnorm` | buy 不变笔数占 buy 总笔数比 | `ntrade_flat_buy / (ntrade_buy + ε)` |
| 27 | `ntrade_buy_updown_imbalance_udnorm` | buy 涨跌笔数不平衡度 | `(ntrade_up_buy - ntrade_down_buy) / (ntrade_buy + ε)` |
| 28 | `ntrade_buy_updownflat_vol_udnorm` | buy 涨跌活跃度 | `(ntrade_up_buy + ntrade_down_buy - ntrade_flat_buy) / (ntrade_buy + ε)` |

#### 组 3：sell 方向成交涨跌统计 [ntrade_sell, ntrade_up_sell, ntrade_down_sell, ntrade_flat_sell]

| # | 列名 | 含义 | 计算公式 |

| 29 | `ntrade_sell_up_udnorm` | sell 上涨笔数占 sell 总笔数比 | `ntrade_up_sell / (ntrade_sell + ε)` |
| 30 | `ntrade_sell_down_udnorm` | sell 下跌笔数占 sell 总笔数比 | `ntrade_down_sell / (ntrade_sell + ε)` |
| 31 | `ntrade_sell_flat_udnorm` | sell 不变笔数占 sell 总笔数比 | `ntrade_flat_sell / (ntrade_sell + ε)` |
| 32 | `ntrade_sell_updown_imbalance_udnorm` | sell 涨跌笔数不平衡度 | `(ntrade_up_sell - ntrade_down_sell) / (ntrade_sell + ε)` |
| 33 | `ntrade_sell_updownflat_vol_udnorm` | sell 涨跌活跃度 | `(ntrade_up_sell + ntrade_down_sell - ntrade_flat_sell) / (ntrade_sell + ε)` |

### 2.2 三分量组 `[all, up, down]`（6 组 × 3 列 = 18 列）

## 三、Bid/Ask 归一化（18 列）

对包含 bid 和 ask 子分量的列组做归一化。

### 3.1 三分量组 `[all, bid, ask]`（1 组 × 3 列 = 3 列）

每组生成 3 列：
- `{all}_bid_abnorm = bid / (all + ε)`
- `{all}_ask_abnorm = ask / (all + ε)`
- `{all}_askbid_imbalance_abnorm = (bid - ask) / (all + ε)`

#### 组 1：quote 变化总数 [nquote, nquote_bid, nquote_ask]

| # | 列名 | 含义 | 计算公式 |
|---|------|------|----------|
| 52 | `nquote_bid_abnorm` | bid 变化次数占 quote 总变化次数比 | `nquote_bid / (nquote + ε)` |
| 53 | `nquote_ask_abnorm` | ask 变化次数占 quote 总变化次数比 | `nquote_ask / (nquote + ε)` |
| 54 | `nquote_askbid_imbalance_abnorm` | bid/ask 变化次数不平衡度 | `(nquote_bid - nquote_ask) / (nquote + ε)` |

### 3.2 二分量组 `[bid, ask]`（5 组 × 3 列 = 15 列）

每组生成 3 列：
- `{bid}_abnorm = bid / (bid + ask + ε)`
- `{ask}_abnorm = ask / (bid + ask + ε)`
- `{ask}_bid_imbalance_abnorm = (bid - ask) / (bid + ask + ε)`

#### 组 2：bid/ask 价格上涨次数 [nquote_bid_up, nquote_ask_up]

| # | 列名 | 含义 | 计算公式 |
|---|------|------|----------|
| 55 | `nquote_bid_up_abnorm` | bid 上涨次数占 bid/ask 上涨总次数比 | `nquote_bid_up / (nquote_bid_up + nquote_ask_up + ε)` |
| 56 | `nquote_ask_up_abnorm` | ask 上涨次数占 bid/ask 上涨总次数比 | `nquote_ask_up / (nquote_bid_up + nquote_ask_up + ε)` |
| 57 | `nquote_ask_up_bid_imbalance_abnorm` | bid/ask 上涨次数不平衡度 | `(nquote_bid_up - nquote_ask_up) / (nquote_bid_up + nquote_ask_up + ε)` |

#### 组 3：bid/ask 价格下跌次数 [nquote_bid_down, nquote_ask_down]

| # | 列名 | 含义 | 计算公式 |
|---|------|------|----------|
| 58 | `nquote_bid_down_abnorm` | bid 下跌次数占 bid/ask 下跌总次数比 | `nquote_bid_down / (nquote_bid_down + nquote_ask_down + ε)` |
| 59 | `nquote_ask_down_abnorm` | ask 下跌次数占 bid/ask 下跌总次数比 | `nquote_ask_down / (nquote_bid_down + nquote_ask_down + ε)` |
| 60 | `nquote_ask_down_bid_imbalance_abnorm` | bid/ask 下跌次数不平衡度 | `(nquote_bid_down - nquote_ask_down) / (nquote_bid_down + nquote_ask_down + ε)` |

#### 组 4：bid/ask 挂单量变化次数 [nquote_bidsize, nquote_asksize]

| # | 列名 | 含义 | 计算公式 |
|---|------|------|----------|
| 61 | `nquote_bidsize_abnorm` | bid 挂单量变化次数占 bid/ask 变化总次数比 | `nquote_bidsize / (nquote_bidsize + nquote_asksize + ε)` |
| 62 | `nquote_asksize_abnorm` | ask 挂单量变化次数占 bid/ask 变化总次数比 | `nquote_asksize / (nquote_bidsize + nquote_asksize + ε)` |
| 63 | `nquote_asksize_bid_imbalance_abnorm` | bid/ask 挂单量变化次数不平衡度 | `(nquote_bidsize - nquote_asksize) / (nquote_bidsize + nquote_asksize + ε)` |

#### 组 5：bid/ask 挂单量增加次数 [nquote_bidsize_up, nquote_asksize_up]

| # | 列名 | 含义 | 计算公式 |
|---|------|------|----------|
| 64 | `nquote_bidsize_up_abnorm` | bid 挂单量增加次数占 bid/ask 增加总次数比 | `nquote_bidsize_up / (nquote_bidsize_up + nquote_asksize_up + ε)` |
| 65 | `nquote_asksize_up_abnorm` | ask 挂单量增加次数占 bid/ask 增加总次数比 | `nquote_asksize_up / (nquote_bidsize_up + nquote_asksize_up + ε)` |
| 66 | `nquote_asksize_up_bid_imbalance_abnorm` | bid/ask 挂单量增加次数不平衡度 | `(nquote_bidsize_up - nquote_asksize_up) / (nquote_bidsize_up + nquote_asksize_up + ε)` |

#### 组 6：bid/ask 挂单量减少次数 [nquote_bidsize_down, nquote_asksize_down]

| # | 列名 | 含义 | 计算公式 |
|---|------|------|----------|
| 67 | `nquote_bidsize_down_abnorm` | bid 挂单量减少次数占 bid/ask 减少总次数比 | `nquote_bidsize_down / (nquote_bidsize_down + nquote_asksize_down + ε)` |
| 68 | `nquote_asksize_down_abnorm` | ask 挂单量减少次数占 bid/ask 减少总次数比 | `nquote_asksize_down / (nquote_bidsize_down + nquote_asksize_down + ε)` |
| 69 | `nquote_asksize_down_bid_imbalance_abnorm` | bid/ask 挂单量减少次数不平衡度 | `(nquote_bidsize_down - nquote_asksize_down) / (nquote_bidsize_down + nquote_asksize_down + ε)` |

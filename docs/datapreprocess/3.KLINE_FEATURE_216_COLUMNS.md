# CROSS_SECTION/KLINE_FEATURE：216 个 K 线形态特征

代码位置：
- `data_preprocess/operator_futures/cross_section/create_feature.py`（入口）
- `data_preprocess/operator_futures/cross_section/base_feature_util.py`（核心计算函数 `process_k_line_feature`）

输入为 `BASE_FEATURE` 的 112 列，通过正则 `r"^(.*?)(open|high|low|close|twap|awap|vwap)(.*)$"` 匹配出含 OHLC 的列组，对每组生成 K 线形态特征。

---

## 统一记号

对每组 OHLC 列，记：
- `O` = open 值
- `H` = high 值
- `L` = low 值
- `C` = close 值
- `TWAP` = twap 值
- `AWAP` = awap 值
- `VWAP` = vwap 值（仅 trades 全量/buy/sell 三组有）

防零常数：`ε = 1e-15`

---

## 通用公式模板

以下 21 个公式适用于有 VWAP 的组（trades origin/buy/sell），无 VWAP 的组（9 个 quote 组）不生成最后 4 个 vwap 相关列，每组 17 个。

| 特征名后缀 | 含义 | 计算公式 |
|-----------|------|----------|
| `klen` | K 线长度（振幅） | `(H - L) / (O + ε)` |
| `kmid` | K 线实体中心偏移 | `(C - O) / (O + ε)` |
| `kmid2` | K 线实体占振幅比 | `(C - O) / (H - L + ε)` |
| `kup` | 上影线长度 | `(H - max(O, C)) / (O + ε)` |
| `kup2` | 上影线占振幅比 | `(H - max(O, C)) / (H - L + ε)` |
| `klow` | 下影线长度 | `(min(O, C) - L) / (O + ε)` |
| `klow2` | 下影线占振幅比 | `(min(O, C) - L) / (H - L + ε)` |
| `ksft` | 收盘价偏移中心 | `(2C - H - L) / (O + ε)` |
| `ksft2` | 收盘价偏移中心（振幅归一化） | `(2C - H - L) / (H - L + ε)` |
| `kotwap` | 开盘价 vs TWAP 偏移 | `(O - TWAP) / (O + ε)` |
| `kotwap2` | 开盘价 vs TWAP 偏移（振幅归一化） | `(O - TWAP) / (H - L + ε)` |
| `kctwap` | 收盘价 vs TWAP 偏移 | `(C - TWAP) / (C + ε)` |
| `kctwap2` | 收盘价 vs TWAP 偏移（振幅归一化） | `(C - TWAP) / (H - L + ε)` |
| `koawap` | 开盘价 vs AWAP 偏移 | `(O - AWAP) / (O + ε)` |
| `koawap2` | 开盘价 vs AWAP 偏移（振幅归一化） | `(O - AWAP) / (H - L + ε)` |
| `kcawap` | 收盘价 vs AWAP 偏移 | `(C - AWAP) / (C + ε)` |
| `kcawap2` | 收盘价 vs AWAP 偏移（振幅归一化） | `(C - AWAP) / (H - L + ε)` |
| `kovwap` | 开盘价 vs VWAP 偏移 | `(O - VWAP) / (O + ε)` |
| `kovwap2` | 开盘价 vs VWAP 偏移（振幅归一化） | `(O - VWAP) / (H - L + ε)` |
